# Personal Portfolio (React + Vite)

A minimal, Apple-inspired portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project structure

- `src/sections/*` – Page sections (Hero, About, Experience, Projects, Skills, Contact)
- `src/components/*` – Reusable UI components (Navbar, cards, layout helpers)
- `src/data/resume.ts` – Central data source for all content (name, experience, projects, skills, contact)

---

> This project is designed to be easy to customize: update `src/data/resume.ts` with your own information and the site updates automatically.
